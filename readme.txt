A customizable chat button plugin with multi-user and group chat support.

== Description ==
Adds a customizable chat button with multi-user and group chat support. Easily configure multiple chat users and display a chat button on your site.

== Installation ==
1. Upload the plugin to your WordPress site.  
2. Activate the plugin through the 'Plugins' menu in WordPress.  
3. Go to 'Swarnim Multi-Chat' to configure the plugin.

== Frequently Asked Questions ==

= How do I add chat users? =

1. Navigate to the plugin settings page under 'Settings' -> 'WhatsApp Multi-Chat'.  
2. Enter the user details, such as name, phone number, and default message.
